# 企业数字化转型指数查询系统

这是一个基于Streamlit的企业数字化转型指数查询系统，用于分析和可视化企业的数字化转型数据。

## 功能特点

1. **数据概览**：展示数据的基本统计信息
2. **维度相关性热力图**：展示不同维度之间的相关性
3. **指数分布**：展示数字化转型指数的分布情况
4. **地理分布**：展示企业的地理分布情况
5. **热力分布**：展示数字化转型指数的热力分布
6. **企业查询**：支持按股票代码和年份查询企业数据
7. **趋势分析**：展示企业历年数字化转型指数的变化趋势
8. **词频分析**：分析企业年报中数字技术词汇的使用情况

## 依赖环境

- Python 3.8+
- Streamlit
- Pandas
- Matplotlib
- Seaborn
- Folium
- Numpy

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用说明（Windows系统）

### 步骤1：打开命令提示符

1. 按下 `Win + R` 键打开「运行」对话框
2. 输入 `cmd` 并按下回车键，打开命令提示符窗口

### 步骤2：导航到项目目录

在命令提示符窗口中输入以下命令，进入项目所在目录：

```bash
cd c:\Users\35047\Desktop\dddddddd
```

### 步骤3：运行Streamlit应用

输入以下命令启动应用程序：

```bash
streamlit run digital_transformation_app.py
```

### 步骤4：打开网页

应用启动后，命令提示符窗口会显示一个本地URL（通常是 http://localhost:8501），复制该URL并粘贴到浏览器地址栏中访问。

## 故障排除

1. **依赖安装失败**：确保Python环境正确配置，尝试使用 `pip install --upgrade pip` 升级pip后重新安装依赖

2. **数据加载失败**：确保「两版合并后的年报数据_完整版.xlsx」文件存在于项目根目录

3. **网页无法访问**：检查命令提示符窗口中显示的URL是否正确，尝试重新启动应用程序

4. **中文显示问题**：应用程序已配置了中文字体支持，如仍有显示问题，请确保Windows系统已安装常用中文字体（如宋体、微软雅黑等）

## 注意事项

- 首次运行时，Streamlit可能需要下载一些额外的资源
- 应用程序使用的是示例经纬度数据，实际使用时可以替换为真实的地理数据
- 数据加载时间可能会因数据量大小而有所不同
